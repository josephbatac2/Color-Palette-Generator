# Color Palette Generator

A professional-grade color palette generator with advanced harmony algorithms, accessibility checking, and color blindness simulation. Built with React, TypeScript, and modern web technologies.

## ✨ Features

### 🎨 Advanced Color Generation
- **6 Color Harmony Types**: Monochromatic, Analogous, Complementary, Triadic, Tetradic, and Split-Complementary
- **Enhanced Color Picker**: Large, prominent color picker with glowing background and visual feedback
- **Real-time Preview**: Instant color updates with smooth animations
- **Multiple Input Methods**: Native color picker, HEX input, and HSL sliders

### 🌈 Curated Palette Collection
- **160+ Professional Palettes** across 8 categories
- **Industry-Inspired**: Web, Brand, Nature, Vintage, Modern, Pastel, Bold, and Monochrome themes
- **One-Click Selection**: Instantly load any curated palette for customization

### ♿ Accessibility & Inclusion
- **WCAG Compliance Checker**: Automatic contrast ratio analysis
- **Color Blindness Simulator**: Test palettes for Protanopia, Deuteranopia, Tritanopia, and Achromatopsia
- **Accessibility Reports**: Detailed breakdown of color pair compliance levels

### 💾 Palette Management
- **Save & Organize**: Personal palette collections with persistent storage
- **Export Options**: CSS variables, SCSS, JSON formats
- **Lightbox View**: Full-screen color inspection with detailed color information
- **One-Click Copy**: Copy HEX, RGB, or HSL values instantly

### 🎯 User Experience
- **Glass Morphism UI**: Modern, translucent design with backdrop blur effects
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Micro-interactions and hover effects throughout
- **Intuitive Navigation**: Tab-based interface with clear visual hierarchy

## 🎨 Color Theory Implementation

### Harmony Algorithms
- **Monochromatic**: Variations in lightness of a single hue
- **Analogous**: Colors adjacent on the color wheel (30° intervals)
- **Complementary**: Colors opposite on the color wheel (180° apart)
- **Triadic**: Three colors evenly spaced (120° intervals)
- **Tetradic**: Four colors forming a rectangle on the color wheel
- **Split-Complementary**: Base color plus two colors adjacent to its complement

### Accessibility Standards
- **AAA Level**: 7:1 contrast ratio (highest standard)
- **AA Level**: 4.5:1 contrast ratio (minimum for normal text)
- **Large Text**: 3:1 contrast ratio for 18pt+ or 14pt+ bold text
